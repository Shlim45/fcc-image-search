Image Search Abstraction Layer

  1. use https://heady-swoop.glitch.me/api/imagesearch/<SEARCH TERM>?offset=<PAGE NUMBER> to perform a Google image search
    • SEARCH TERM is the image keywords you wish to search for
    • PAGE NUMBER is which page of the results you wish to see
  2. use https://heady-swoop.glitch.me/api/latest/imagesearch/ to view the last 10 search queries
